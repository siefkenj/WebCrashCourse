Lesson 1: The DOM

DOM stands for Document Object Model and is the internal representation of a web page. If you want to make a dynamic change to a web page, you will use a browser-integrated language (like Javascript) to make a change to the DOM. The web browser automatically detects any changes to the DOM and will update the webpage accordingly.

The DOM is a tree structure. Every node has one parent and potentially many children. When you write HTML, you are explicitly defining all the nodes and their hierarchical relationships. Nodes/Elements

I will use the words Node and Element interchangeably. The word Node is typically associated with a tree structure and the word Element is typically used when discussing the nodes in the tree structure specified by the DOM.

An HTML element (almost the same thing as a DOM element) is called a tag and is specified as <blabla prop1="xxx" prop2="yyy">...list of children...</blabla>. The element has three components: the tag name, attributes, and content. For the example given, these break down as

  • Tag Name: blabla
  • Attributes: prop1, prop2
  • Content: ...list of children...

Special Attributes

HTML tags have several special attributes. Special for every tag is id, class, and style. Specific tags, for instance <a> have other special attributes, like href, src, etc..

The universally special attributes id and class can be used to search for and identify specific elements in the DOM. The tags should be specified as follows:

  • id: unique identifier for an element consisting of the characters a-z (capital or lower case) and - (hyphen).
  • class: a space-separated list of identifiers consisting of the characters a-z (capital or lower case) and - (hyphen).

Tag Name, class, and id play a special role in what is called a selector (or CSS selector string), which is a way to specify a pattern representing a range of elements. For example, to select elements with Tag Name of p and an id of foo, the selector string would be p#foo. To select elements with Tag Name of p and and class of bar, the selector string would be p.bar.

If you were to just use the selector string .bar, it would select all elements of class bar no matter what the tag name is.
